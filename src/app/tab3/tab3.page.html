About AMY Capture Now

It's a camera application which allows the user to take picture from back as well as front camera. The user can turn on/off/set to auto flashlight while taking the picture. The app also allows the user to upload picture from gallery.


Plugins Used

  • Camera

    To capture the image

  • FileSystem

    To save the clicked picture to the device.

    To load image to the applcation from the device
